ROM90201.BIN is a dump of the 90201 printer interface card ROM.
The memory address starts at 0x7800 (30720) and it is 2K in size.

Apparantly there were 2 printer interface cards for the BIT90:
1. A universal centronics parallel card.
2. The 90201 parallel card for the 40 column BIT90 printer/plotter.
